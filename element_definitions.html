Chapters 1 - 3 element definitions

Each HTML element tells the browser something about the information that sits between its opening and closing tags.

<h1> </h1>
This is the main heading element.
<h2> </h2>
This is a level 2 heading element.
<h3> </h3>
This is a level 3 heading element.
<h4> </h4>
This is a level 4 heading element.
<h5> </h5>
This is a level 5 heading element.
<h6> </h6>
This is a level 6 heading element.
<p> </p>
This element creates a paragraph, with an opening tag and the beginning and a closing tag at the end of the paragraph.
<b> </b>
Enclosing characters or words between this tag, they will appear bold.
<i> </i>
Enclosing characters or words between this tag, they will appear italic.
<sup> </sup>
This element is used to superscript characters.
<sub> </sub>
This element is used to subscript characters.
<br />
This element is used if you need to add a new line inside a paragraph.
<hr />
This element is used to create a horizontal line between paragraphs.
<strong> </strong>
This element is used if you want to add strong emphasis to characters or words in a paragraph.
<em> </em>
This element is used if you want to add slight emphasis to characters or words in a paragraph.
<blockquote> </blockquote>
This element is used if you want to quote sentences or paragraphs.
<q> </q>
This element is used if you want to quote something with a sentence or paragraph.
<abbr> </abbr>
This element is used to abbreviate words, or use an acronym.
<cite> </cite>
This element is used if you are referencing another person's work.
<dfn> </dfn>
This element is used define a word in a document.
<address> </address>
This element is used to contain contact information, whether it is just a name, email address, or physical address.
<ins> </ins>
This element is used to show information added into a document.
<del> </del>
This element is used to show deleted information from a document.
<s> </s>
This element is used to show information that is no longer relevant, however the information should not be deleted
<ol> </ol>
This element creates an ordered list.
<ul> </ul>
This element creates an unordered list.
<li> </li>
This element is used for both ordered and unordered lists. The items between these elements are the list items.
<dl> </dl>
This element creates a definition list.
<dt> </dt>
This element is used to contain the term you are going to be defining.
<dd> </dd>
This element is used to contain the definition of the above term.
